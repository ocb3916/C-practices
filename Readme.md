<h3 align="center">
예약시스템 만들기
</h3><br><br>
<p>
 본 의뢰서는 오차해를 갑으로 지칭하며 오찬빈을 을로 지칭한다.<br>
갑은 을에게 'C를 기반으로 만드는 호실 예약 시스템' 의뢰서이다.<br>
의뢰서의 내용은 다음과 같다.
</p><br><br><p>
 1. 예약시스템의 인사말은 '어서오세요! 오가네 별장입니다.' 이다.
 2. 인사말 후 바로 밑에 기능 선택 문구가 나온다.
 3. 기능 선택 문구는 다음과 같다.
	<ul>
		<li>기능 메뉴</li>
		<li>1. 예약</li>
		<li>2. 예약 확인</li>
		<li>3. 종료 </li>
	</ul>
 4. 예약시스템의 기본 기능은 '입출력'으로 입력은 5, 출력은 6 내용과 같다.<br>
